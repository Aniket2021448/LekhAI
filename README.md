# LekhAI
Smart data-entry automation app that converts handwritten slips into structured records using OCR and syncs them with Google Sheets.

google ml kit text recognition package
image picker package

get latest version in your flutter project

Run android studio and go to virtual device manager to start the android emulator. and wait for it to start completely

after that run "flutter run"


Home Screen
 ├─ Open Camera → Capture Image → Review Screen → OCR → LLM Structuring → Save Slip
 ├─ Open Gallery → Pick Image → Review Screen → OCR → LLM Structuring → Save Slip
 └─ View Saved Slips
       └─ Folder List (Grouped by Customer / Vendor Name)
             └─ Slip List (Sorted by Date)
                   └─ Slip Detail (Photo + Parsed Data Display)
